SelectC8 jQuery(v1.0.0)

SelectC8 es una librería que hace el uso del elemento select más sencillo.

con la configuración de propiedades y manejo de eventos puedes ahorrar tiempo en la ejecución de tu funcionalidad y evitas tener las mismas líneas de código por toda tu aplicación.

La idea base de este componente es llenar de manera dinámica los controles select en tu aplicación, centralizando código y haciendo tus llamadas más limpias, puedes utilizarlo dentro de peticiones ajax, o en tu funciones con json listos para ser utilizados.

Instalación

Realiza la referencia de SelectC8 en tu proyecto la cual la puedes obtener desde el repositorio de github Select-C8 jQuery

<script src="SelectC8jQuery.min.js"></script>

Documentación del componente

La siguiente configuración le vida a la librería selectC8 en la cual podrás manejar eventos y propiedades facilitando la utilización de comportamientos de tu select.

Configuración

data: []

Array de objetos con el cual SlectC8 generará las opciones en tu select, puede ser compuesto por un simple array de strings o un array de de objetos con propiedades para poblar tu control. Es un valor requerido.

Ejemplo.
 let fruits = [{id: 1,name: "Apple"},{id: 2,name: "Banana"},{id: 3,name: "Grapes"},{id: 4,name: "Watermelon"}];
 let fruits = [{id: 1,name: "Apple"},{id: 2,name: "Banana"}];
valueProperty: "id", textProperty: "name"

Estos dos valores al momento de enviar un array definen cuál será tu value y text en los option de tu control, es importante tomarlos en cuenta cuando se manda un array complex o personalizado. 

dataExtended: null

Propiedad que define un atributo extra en cada option de tu control, si requieres un valor adicional a value y text, esta opción te brinda el manejo de una propiedad extra. 

selectedValue: null

configuración la cual te ayuda a colocar de manera predeterminada una opción seleccionada al momento de hacer realizar la carga inicial del control.

assignWithoutProperties: false

configuración que determina si el arreglo que envías en la propiedad data utilizará propiedades o no. 

 defaultFirstOptionValue: "", defaultFirstOptionText: "-- Select option --"

Estos dos valores al momento de inicializar tu control SelectC8 asignan un valor inicial en tu select. 

onChange

Defines con una función de tipo callback lo acción que deseas hacer cada vez que cambies el valor seleccionado en tu select. 

onSuccess

Función de tipo callback la cual te permite realizar algun evento una vez que el control se renderice. 

            
        let defaults = {
            data: null,
            valueProperty: "",
            textProperty: "",
            dataExtended: null,
            selectedValue: null,
            assignWithoutProperties: false,
            appendDefaultFirstOption: true,
            defaultFirstOptionValue: "",
            defaultFirstOptionText: "-- Select option --",
            onChange: null,
            onSuccess: function () {
                return false
            }
        };

Demo

Se describe como puedes utilizar el control SelectC8 en tu proyecto

Ejemplo 1

 <select class="custom-select" name="example1" id="example1"></select>
             
      let $example1 = $("#example1");
      const fruits = [{
                    id: 1,
                    name: "Apple"
                },
                {
                    id: 2,
                    name: "Banana"
                },
                {
                    id: 3,
                    name: "Grapes"
                },
                {
                    id: 4,
                    name: "Watermelon"
                }
            ];
     $example1.SelectC8({
                data: fruits,
                valueProperty: "id",
                textProperty: "name",
            });

Ejemplo 2
